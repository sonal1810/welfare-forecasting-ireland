# Forecasting Welfare Recipients in Ireland

## Project Overview

This project is part of my MSc Data Analytics thesis. The aim is to analyse welfare recipient trends in Ireland and build forecasting models to predict future recipient numbers across schemes and counties.

The project focuses on data modelling and prediction using Irish public datasets.

## Objectives

- Analyse welfare recipient trends in Ireland
- Perform exploratory data analysis
- Build forecasting models
- Compare model performance
- Present insights using clear visualisations

## Dataset

The project uses publicly available Irish social protection and welfare-related datasets, including data from data.gov.ie and official annual statistics reports.

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Statsmodels
- SQL
- Power BI
